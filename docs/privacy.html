1. Introdução

Esta Política de Privacidade descreve como o aplicativo Biblioteca BMSQ, desenvolvido por Adilson Manoel do Nascimento, coleta, utiliza e protege as informações dos usuários da Biblioteca Marlene de Souza Queiroz da E. E. Cel. José Venâncio de Souza.

Não coletamos dados de localização, não exibimos publicidade e não rastreamos você fora do aplicativo. Seus dados são usados exclusivamente para o funcionamento da biblioteca escolar.

2. Dados coletados

Para utilizar o aplicativo, coletamos apenas as informações necessárias para identificar o usuário e gerenciar os empréstimos:

Dado Finalidade Obrigatório
Nome completo Identificação no sistema da biblioteca Sim
Endereço de e-mail Login e comunicações da biblioteca Sim
ID de usuário Identificador interno gerado automaticamente Sim
Matrícula (alunos) Vinculação com o registro escolar Sim (alunos)
Turma (alunos) Organização por turma na biblioteca Sim (alunos)
Disciplina (professores) Identificação do professor no sistema Sim (professores)
Token de notificação (push token) Envio de notificações da biblioteca (comunicados, avisos) Não (requer permissão do dispositivo)

Não coletamos: localização, contatos do telefone, dados financeiros, informações de saúde, histórico de navegação, dados biométricos ou qualquer identificador de publicidade.

3. Uso da câmera

O aplicativo solicita acesso à câmera exclusivamente para fotografar capas de livros durante o cadastro de obras no acervo. As imagens não são armazenadas nos servidores — são processadas momentaneamente para identificar o título, autor e demais dados da obra por meio de inteligência artificial.

O acesso à câmera é solicitado apenas no momento do uso e pode ser negado sem impedir as demais funcionalidades do app.

4. Notificações push

O aplicativo pode solicitar permissão para enviar notificações ao seu dispositivo. Essa permissão é opcional — recusá-la não impede o uso de nenhuma funcionalidade do app.

Caso você conceda a permissão, um token de notificação (identificador anônimo gerado pelo serviço Expo) é armazenado associado à sua conta. Ele é usado exclusivamente para enviar comunicados e avisos da biblioteca (como novas aquisições ou prazos de devolução). Não é usado para publicidade ou rastreamento.

Você pode revogar a permissão a qualquer momento nas configurações do seu dispositivo (Ajustes → Biblioteca BMSQ → Notificações). O token também é excluído automaticamente quando você exclui sua conta.

5. Como os dados são armazenados

Todos os dados são armazenados de forma segura em servidores do Supabase (PostgreSQL), com conexão criptografada via SSL/TLS. O acesso ao banco de dados é restrito à API da biblioteca, autenticada por token JWT.

Senhas nunca são armazenadas em texto puro — são protegidas com hash criptográfico (bcrypt).

6. Compartilhamento de dados

Seus dados não são vendidos, alugados ou compartilhados com terceiros para qualquer finalidade comercial, publicitária ou de rastreamento.

Os dados são acessados apenas pelos funcionários autorizados da biblioteca escolar (bibliotecário) para fins de gestão de empréstimos e comunicação com os alunos e professores da escola.

7. Seus direitos

Você tem os seguintes direitos sobre seus dados pessoais:

  • Acesso: solicitar quais dados estão armazenados sobre você.
  • Correção: solicitar a correção de dados incorretos.
  • Exclusão: excluir sua conta e todos os dados associados permanentemente.

8. Como excluir sua conta

Você pode excluir sua conta e todos os seus dados diretamente pelo aplicativo, sem precisar entrar em contato com a biblioteca:

  1. Abra o aplicativo e faça login.
  2. Toque na aba Perfil (ícone 👤 no menu inferior).
  3. Role até o final da página e toque em "Excluir minha conta".
  4. Confirme a exclusão nas duas telas de confirmação.

A exclusão é permanente e imediata. Todos os dados associados à sua conta — histórico de empréstimos, lista de desejos, avaliações e informações de cadastro — são removidos definitivamente dos servidores. Esta ação não pode ser desfeita.

9. Retenção de dados

Os dados são mantidos enquanto a conta do usuário estiver ativa. Ao excluir a conta, todos os dados pessoais são removidos permanentemente dos servidores, sem período de retenção adicional.

10. Segurança

Adotamos medidas técnicas para proteger suas informações, incluindo: comunicação criptografada via HTTPS, armazenamento de senhas com bcrypt, autenticação por token JWT com expiração, e acesso restrito ao banco de dados.

11. Alterações nesta política

Podemos atualizar esta política periodicamente. Quando isso ocorrer, a data de "última atualização" no topo desta página será revisada. Recomendamos que você consulte esta página ocasionalmente.
